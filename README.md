# Land Rover Defender V8 Bond Edition Landing Page

A hyper-premium, cinematic scrollytelling landing page showcasing the pure capability and dark-mode aesthetic of the Land Rover Defender V8 Bond Edition.

## Features

- **Apple-Level Scrollytelling**: Deep integration with scroll events seamlessly manipulating a 240-frame 3D dissection animation.
- **Pure Dark-Mode Aesthetic**: Meticulously crafted dark-mode styling utilizing deep charcoal blacks (`#050505`), brushed gold elements (`#C8922A`), and dark greens (`#1B4D3E`).
- **Hardware-Accelerated performance**: Custom Canvas engine written in Vanilla JavaScript implementing `requestAnimationFrame` ensuring buttery smooth transitions across the continuous `500vh` scrollable scene.
- **Glassmorphism Navigation**: Floating navigation using native Apple-styled backdrop blurring with seamless fade-in interactions.

## Technology Stack

- **HTML5**: For semantic layout and structuring.
- **Vanilla CSS3**: Leveraging complex gradients, custom properties/variables, and hardware-accelerated transforms (`translateY` / `opacity`).
- **Vanilla JavaScript**: Controlling DOM manipulations, scroll syncing formulas, and Canvas 2D rendering.

## Setup & Running

This is a static implementation. To run it locally:
1. Clone the repository.
2. Ensure you have the `ezgif-frame-001.jpg` to `ezgif-frame-240.jpg` assets located in your root directory.
3. Open `index.html` in your favorite modern browser (or run via a simple local live server like `Live Server` plugin in VS Code).
